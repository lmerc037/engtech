Comprehensive Guide for Reimaging a Computer

  • Create a Service Ticket (if not already done by the customer) ?
    Open a ticket with the necessary details. Add the Drop-Off information (visible only internally) and the computer's name. Note any software installation requirements. Ensure that all important files or data are backed up before proceeding with reimaging.
  • Optional Step: Verify Computer in SnipeIT Inventory ?
    Check if the computer is added to SnipeIT for asset management purposes. If not, consider adding it.
  • Verify Compatibility with Windows 11 ?
    Ensure the computer meets the Windows 11 system requirements: TPM 2.0, 8th generation processor or higher, 256 GB of storage, and 8 GB of RAM.
  • Remove the Computer from SCCM ?
    Provide the computer name or service tag to a senior IT member or the IT Lead. Ensure the computer is removed from SCCM before proceeding with the reimaging process.
  • Installation Preparation ?
    Plug the computer into a power source, connect it to a VLAN using a USB-Ethernet dongle, and insert the SCCM USB key into the computer.
  • Installation Process ?
    Boot the computer from the USB key (F12) and follow the instructions to complete the installation. Remove the USB key when prompted, enter the serial number manually, select the appropriate department, and add the prefix (ST-, CH-, AD-).
  • Installation Troubleshooting ?
    Common errors during installation include SCCM-related conflicts or BitLocker issues. If you encounter errors, follow the troubleshooting steps to resolve them.
  • Add Site Administrator as an Admin User ?
    Log into the computer, open "Computer Management," and add a new user with the username `siteadmin` and password `Etudiant`. Uncheck "User must change password at next logon" and check "User cannot change password" and "Password never expires." Then add `siteadmin` to the Administrators group. Add the computer owner as an admin if they are a professor or PhD student. Do not add staff members without supervisor confirmation.
  • Install System and Software Updates ?
    Log in as `siteadmin`, run Dell Command Update, and install any available firmware and software updates. Run Windows Update and install all available updates. Disable MAC Passthrough in BIOS settings.
  • Notify the Client for Pickup ?
    Update the ticket and include the booking link to notify the client that the computer is ready for pickup. Confirm if the client requires any additional software installations before the handover.
  • Final Handover to the Client ?
    At the front desk, connect the computer to the network using a USB Ethernet dongle. Ask the client to log in with their credentials. Verify that the client has appropriate admin rights, if needed, for professors or PhD students.
